## 🏁 Getting Started <a name = "getting_started"></a>

### Prerequisites

Install:

- [NodeJS](https://nodejs.org/en/)

### Installing

1. Install dependencies

```
npm install
```

## 🔧 Running the app <a name = "run"></a>

run:

```
npm run dev

```
